Proformance testing occlusion culling in Unity.

Unity Occlusion Culling is a technique used in the Unity game engine to improve performance by reducing the number of objects that need to be rendered. It works by determining which objects in a scene are not visible to the camera, and then culling (removing) them from rendering. This results in a reduced number of drawcalls being sent to the graphics card and thus it can lead to an increase in proformance.

Unity Occlusion Culling uses a combination of techniques such as frustum culling, occlusion queries, and portal rendering to achieve its results. Frustum culling eliminates objects outside the camera's field of view, occlusion queries check if an object is hidden behind another object, and portal rendering allows culling of objects behind closed doors or other obstructions.
